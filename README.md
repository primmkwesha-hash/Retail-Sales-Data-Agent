## Problem Statement 
Retail busines collect a lot of sales data every day. It can be difficult to analyse this data manually and identify useful business insights.

## Aim of the Project
The aim of this project was to build a Retail Sales Data Agent using Databricks Genie to analyse sales and answer business questions.

## Objectives 
Objectives
1. Upload the retail sales dataset
2. Create a Genie Agent
3. Ask business questions
4. Validate the agent's answers

## Summary of Results 
The Genie Agent answered the business questions successfully. The validation showed that the selected answers matched the data confirming that the agent produced reliable results.

## Tools Used
1. Databricks
2. Databricks Genie
3. SQL
4. Microsoft Word
