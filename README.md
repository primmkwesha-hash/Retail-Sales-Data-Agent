# Retail Sales Data Agent Using Databricks Genie 

## Project Title 
Retail Sales Data Agent Using Databricks Genie 


## Project Objectives 
1. Upload the retail sales dataset
2. Create a Genie Agent
3. Ask business questions
4. Validate the agent's answers

## Summary of Results 
The Genie Agent answered the business questions successfully. The validation showed that the selected answers matched the data confirming that the agent produced reliable results.

## Tools Used
1. Databricks
2. Databricks Genie
3. SQL
4. Microsoft Word
   
   
## Dataset Overview
The uses a retail sales dataset containing:
1. Transaction details
2. Customer information
3. Product categories
4. Purchase quantities
5. Sales and revenue values
6. Customer demographic data

## Steps Followed
• Uploaded the retail sales dataset into Databricks.
• Prepared and reviewed the dataset.
• Created and configured a Databricks Genie Agent.
• Asked business related questions using natural language.
• Validated selected responses using SQL queries and Python.
• Documented the findings.


## Key Insights 
•	The analysis showed that Clothing was one of the strongest performing categories based on transaction volume. 
•	Customer purchasing patterns also showed seasonal changes.
•	Age appeared to have an influence on purchasing behavior while gender had less impact.
•	Some products generated high sales volumes but relatively lower revenue.
  
## Conclusion 
This project successfully demonstrated how a Databricks Genie Agent can be used to analyze retail sales data and answer business questions. The most challenging part was validating the agent’s responses and ensuring the answers matched the dataset. In future, the project could be improved by adding more data sources and creating more advanced analysis.
