### Retail Sales Data Agent Using Databricks Genie 

# Project Title 
Retail Sales Data Agent Using Databricks Genie 


# Project Objectives 
1. Upload the retail sales dataset
2. Create a Genie Agent
3. Ask business questions
4. Validate the agent's answers

# Summary of Results 
The Genie Agent answered the business questions successfully. The validation showed that the selected answers matched the data confirming that the agent produced reliable results.

# Tools Used
1. Databricks
2. Databricks Genie
3. SQL
4. Microsoft Word
   
   
# Dataset Overview
The uses a retail sales dataset containing:
1. Transaction details
2. Customer information
3. Product categories
4. Purchase quantities
5. Sales and revenue values
6. Customer demographic data

# Steps Followed
1. Uploaded the retail sales dataset into Databricks.
2. Prepared and reviewed the dataset.
3. Created and configured a Databricks Genie Agent.
4. Asked business related questions using natural language.
5. Validated selected responses using SQL queries and Python.
6. Documented the findings.
